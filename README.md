# 🤖 AI Desktop Robot - 桌面智能机器人项目

> 一个集成了AI对话、可扩展硬件交互能力的智能桌面机器人项目。项目初期致力于构建核心AI大脑与Web服务，为后续的语音、视觉及实体动作控制奠定基础。

## ✨ 核心特性

- **🧠 智能AI对话**：集成大型语言模型（如DeepSeek），提供自然、流畅的对话体验。
- **🌐 实时Web服务**：基于 FastAPI 构建高性能后端服务器，提供标准的 RESTful API。
- **💻 跨平台访问**：提供简洁的网页聊天界面，可通过浏览器在任何设备上访问。
- **📦 模块化设计**：代码结构清晰，AI客户端、服务器、配置相互分离，易于维护和扩展。
- **🔧 面向硬件扩展**：项目架构为后续集成摄像头、麦克风、扬声器及电机控制等硬件模块预留了接口。
